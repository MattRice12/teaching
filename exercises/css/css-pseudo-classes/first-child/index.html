:first-child & :nth-child()

We've discussed a few CSS pseudo-classes before:

  • :hover --> which applies styles to an element when your mouse hovers over that element.
  • :visited --> which applies styles to an a-tag (a link) if that link has been visited.
  • :checked --> which applies styles when a checkbox or radio button is selected.

Today, let's discuss the pseudoclasses :first-child and :nth-child. This pseudoclass is appended to an element or class when there are a group of the same elements or classes in that section. Only the first or nth child receives the proceeding styles.

:first-child

This applies the style only to the first element in the group.

<ul>
<li>Item 1</i>
<li>Item 2</i>
<li>Item 3</i>
<li>Item 4</i>
<ul />

ul li:first-child {
color: red;
}

Result:
  • Item 1
  • Item 2
  • Item 3
  • Item 4

:nth-child

This applies styles to the nth item in the group, where n is the argument you pass into the style. Using same list from above, if we say :nth-child(2), we can select the second item in the group.

ul li:nth-child(2) {
color: red;
}

Result:
  • Item 1
  • Item 2
  • Item 3
  • Item 4

But with :nth-child, we can get even fancier. :nth-child accepts n itself as an argument. Doing so would select every element in the list (not too useful). However, if we pass in 2n, now we're selecting every other element in the group. Much more useful.

ul li:nth-child(2n) {
color: red;
}

Result:
  • Item 1
  • Item 2
  • Item 3
  • Item 4

(2n) allows us to choose every even row in the group. We can also accomplish this by calling :nth-child(even). If we want to choose the odd row, we can either call :nth-child(2n-1) or :nth-child(odd).

This is useful when presenting tables. Staggering the background colors helps the reader's eye keep track.

ul li:nth-child(2n) {
background: beige;
}
ul li:nth-child(2n-1) {
background: white;
}

Result:
  • Item 1
  • Item 2
  • Item 3
  • Item 4

:last-child & :nth-last-child()

Like :first-child and :nth-child, :last-child and :nth-last-child select specific items in the group, but from the other direction.

ul li:last-child {
color: blue;
}

Result:
  • Item 1
  • Item 2
  • Item 3
  • Item 4


ul li:nth-last-child(2n) {
background: beige;
}
ul li:nth-last-child(2n-1) {
background: white;
}

Result:
  • Item 1
  • Item 2
  • Item 3
  • Item 4
