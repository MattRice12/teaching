:checked

We've discussed a few CSS pseudo-classes before:

  • :hover --> which applies styles to an element when your mouse hovers over that element.
  • :visited --> which applies styles to an a-tag (a link) if that link has been visited.

Today, let's discuss the pseudoclass :checked. This applies to an element, typically a radio button or checkbox, that is selected.

Radio buttons and checkboxes have 2 possible values, either checked or not checked. In other words, their values are binary. CSS let's us take advantage of these predictable states.

When the element is checked, we can tell it to do one thing

input[type="radio"]:checked {
/* some styling */
}

Using the :not pseudo-class, we can tell the element to do somethign else:

input[type="radio"]:not(:checked) {
/* some styling */
}

Additionally, using the CSS sibling selectors (~ and +), the child selector (>), or the descendant selector (space), we can affect the neighboring elements based on the radio or checkbox's state:

div {
display: none;
}

input[type="checkbox"]:checked ~ div {
display: block;
}

We should use the elements' classes to more specifically target the content we want

.nav {
display: none;
}

.nav-checkbox:checked ~ .nav {
display: block;
}

Helpful Styling Info

1) Hide the Button

To hide the default radio button and to use a custom icon, we need to:

  1. link a label to the radio or checkbox, and
  2. set the checkbox display to none.

To link the label to the radio or checkbox, the "id" attribute on the radio or checkbox and the "for" attribute on the label need to match and need to be unique (meaning this specific name isn't used elsewhere). This link lets the label find it's associated radio or checkbox.

By connecting the label to the checkbox, we don't actually need the checkbox present to check or uncheck it. Clicking the label will do the same thing.

<input type="checkbox" class="nav-checkbox" id="navHamburger" />
<label class="nav-menu-label" for="navHamburger">
<i class="fas fa-bars"></i>
</label>

.nav-checkbox {
display: none;
}

2) Icon

My nav icon is taken from the font-awesome library. I set the icon as the label by inserting it between the opening and closing label tags.

<label>
<i class="fas fa-bars"></i>
</label>

3) Rotate the Icon

Lastly, to get my icon to rotate, I used the CSS property "rotate". I also used a transition of .2 to smooth out the rotation:

.nav-checkbox:checked ~ label .fas {
transform: rotate(90deg);
transition: .2s;
}

.nav-checkbox:not(:checked) ~ label .fas {
transform: rotate(0deg);
transition: .2s;
}
