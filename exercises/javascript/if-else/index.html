Overview

If/else statements are another fundamental part of any sort of programming. These allow us to receive different outputs based on the input.

if (condition) {
return result1;
} else {
return result2;
};

In the above example, we begin with the line if (condition). If the condition is truthy, then the code inside the if statement executes. If the condition is falsey then, in this example, the code inside the else statement executes.

A value or expression is truthy if it evaluates to true in a Boolean context. We can manually demonstrate this converting the value to a boolean with Boolean(value).

Examples of truthy and falsey values:

Boolean(true);
// true
Boolean(false);
// false
Boolean(1);
// true
Boolean(0);
// false
Boolean("");
// false
Boolean("Hello");
// true
Boolean(1 > 2);
// false
Boolean(2 === 2);
// false

With this in mind, we can build our if/else statements with intention. In the following example, I'm going to build an if/else statement that returns the higher value of two numbers.

let a = 2;
let b = 3;

if (a > b) {
return a;
} else {
return b;
};

In this example, the conditional evaluates to false, so the code in the else block executes. Let's put this inside a function called max(a, b) and console.log the results.

function max(a, b) {
if (a > b) {
return a;
} else {
return b;
};
};

console.log(max(2, 3));
// 3

Else If

else if is another component of an if statement. The code in the else if section excutes only if (1) the if condition is falsey and (2) the else if statement is truthy.

In the following example, the bouncer function returns a response based on the input age.

function bouncer(age) {
if (age < 18) {
return "You're too young to enter";
} else if (age >= 18 && age < 21) {
return "You're old enough to enter but you're too young to drink.";
} else {
return "You're old enough enter and have drinks.";
};
};

let age = prompt("How old are you?");
console.log(bouncer(age));

If statements can have as many else if blocks as needed. That being said, the more else ifs you have, the more likely you should be using something besides an if/else statement, or to refactor the current statement into something less sprawling. On the other hand sprawling if/else statments are sometimes unavoidable.

The Even Shorter Solution without obfuscating too much
const seasonObject = {
"Winter": ["december", "january", "february"],
"Spring": ["march", "april", "may"],
"Summer": ["june", "july", "august"],
"Fall": ["august", "september", "october"]
};

const seasonFinder = (month) => {
let season = Object.keys(seasonObject).filter(season => {
return seasonObject[season].includes(month.toLowerCase());
});
if (season.length > 0) {
return `It's ${season}!`;
} else {
month = prompt(`"${month}" is not a valid input. What month is it?`);
return seasonFinder(month);
};
};

let month = prompt("What month is it?");
console.log(seasonFinder(month));
